# 참고 이미지 라이브러리

Virtual Try-On을 위한 참고 이미지 라이브러리입니다.

## 📁 디렉토리 구조

```
reference-library/
├── {브랜드명}/              # 브랜드별 폴더
│   ├── dress/              # 원피스/드레스
│   ├── top/                # 상의/블라우스
│   ├── bottom/             # 하의/바지
│   └── outer/              # 아우터/재킷
└── README.md
```

## 📋 파일 명명 규칙

```
{브랜드}_{카테고리}_{번호}_{설명}.jpg
```

예시:
- `lusida_dress_001_sleeveless.jpg` - 루시다 민소매 원피스
- `lusida_top_001_short_sleeve.jpg` - 루시다 반소매 상의

## 🏷️ 카테고리 분류 기준

### dress (원피스/드레스)
- 원피스, 드레스, 롬퍼
- 상하의가 연결된 의류

### top (상의/블라우스)
- 블라우스, 티셔츠, 셔츠, 니트
- 상의만 있는 의류

### bottom (하의/바지)
- 바지, 치마, 슬랙스
- 하의만 있는 의류

### outer (아우터/재킷)
- 재킷, 코트, 가디건, 점퍼
- 겉옷

## 📝 사용 방법

### 1. 새 이미지 추가
```bash
# 브랜드 폴더 생성
mkdir -p reference-library/{브랜드명}/{카테고리}

# 이미지 저장
cp 이미지.jpg reference-library/{브랜드명}/{카테고리}/
```

### 2. 이미지 선택 기준
✅ **좋은 참고 이미지:**
- 모델이 정면을 보고 있음
- 전신이 잘 보임
- 배경이 깔끔함
- 옷의 디테일이 선명함
- 포즈가 자연스러움

❌ **피해야 할 이미지:**
- 옷이 잘려 보임
- 과도한 액세서리
- 복잡한 배경
- 흐릿한 이미지

### 3. 의류 타입 매칭 가이드

**중요:** 참고 이미지의 의류 길이/형태가 대상 의류와 비슷해야 합니다!

| 대상 의류 | 적합한 참고 이미지 |
|----------|------------------|
| 반소매 원피스 | 반소매/민소매 원피스 |
| 긴팔 블라우스 | 긴팔 상의 |
| 반바지 | 반바지/짧은 치마 |
| 롱코트 | 롱코트/긴 아우터 |

## 🔧 라이브러리 관리

### 이미지 확인
```bash
# 카테고리별 개수 확인
ls reference-library/lusida/dress/ | wc -l

# 모든 이미지 미리보기
open reference-library/lusida/dress/
```

### 중복 제거
```bash
# 중복 파일 찾기
fdupes -r reference-library/
```
