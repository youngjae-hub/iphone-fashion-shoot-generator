# 의류 제품컷 리터칭 기능 - 핸드리터칭 레퍼런스 분석

## 개요
2025년 1월 22일 핸드리터칭 레퍼런스 이미지 분석 결과를 바탕으로 업데이트되었습니다.

---

## 핸드리터칭 레퍼런스 분석

### 분석한 배치
- 12_1N (제품컷 원본 vs 제품컷 리터칭)
- 11_4N (제품컷 원본 vs 제품컷 리터칭본)
- 11_2N
- 11_1N

### 분석 결과

| 항목 | 핸드리터칭 결과 |
|------|----------------|
| 배경색 | #F8F8F8 (오프화이트) |
| 그림자 | **없음** |
| 목 라인 처리 | 클린 컷 (페더링/페이드 없음) |
| 실루엣 변환 | 원본 형태 그대로 유지 |
| 스케일 | 프레임의 약 78% (상하좌우 여백 ~11%) |
| 위치 | 정중앙 |

### 핵심 발견

1. **그림자 없음**: 핸드리터칭 결과물에는 드롭쉐도우가 전혀 없습니다
2. **페더링 없음**: 목/상단 부분이 배경으로 서서히 사라지지 않습니다
3. **도식화 없음**: 행거컷 → 플랫레이 변환 시도하지 않습니다
4. **단순 누끼**: 배경만 제거하고 깔끔한 배경에 배치하는 것이 전부입니다

---

## 구현 변경사항

### 제거된 기능
- ❌ 타원형 드롭쉐도우
- ❌ 상단 페더링 (목 라인 페이드)
- ❌ SDXL 도식화 변환

### 유지/수정된 기능
- ✅ 배경 제거 (bria/remove-background)
- ✅ #F8F8F8 배경색
- ✅ 스케일 0.78 (레퍼런스 기준)
- ✅ 정중앙 배치

## 처리 파이프라인 (단순화됨)

```
1. 이미지 업로드
   ↓
2. 리사이즈 (최대 1500px)
   ↓
3. [API] 배경 제거 (bria/remove-background)
   ↓
4. [클라이언트] 콘텐츠 바운딩 박스 감지
   ↓
5. [클라이언트] 배경색 적용 (#F8F8F8)
   ↓
6. [클라이언트] 정중앙 배치 (스케일 0.78)
   ↓
7. 최종 이미지 출력 (JPG/PNG)
```

## 향후 개선 가능 사항

현재 구현은 핸드리터칭 결과물과 거의 동일한 결과를 목표로 합니다.
추가 개선이 필요한 경우:

1. **색보정**: 원본 대비 밝기/대비 조정
2. **미세 먼지 제거**: 의류 표면의 작은 이물질 제거
3. **배경 제거 품질**: 털/보풀 경계 처리 개선
