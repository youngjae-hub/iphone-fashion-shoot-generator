## Upload 탭 - 의류 이미지 업로드

### 업로드 방법
- **드래그 앤 드롭**: 이미지 파일을 업로드 영역에 끌어다 놓기
- **클릭**: 업로드 영역을 클릭하여 파일 선택

### 지원 형식
- JPG, PNG, WebP
- 최대 5장 동시 업로드 가능

### 좋은 의류 이미지 조건
| 권장 | 비권장 |
|------|--------|
| 깔끔한 배경 (흰색/단색) | 복잡한 배경 |
| 의류 전체가 보이는 이미지 | 일부만 보이는 이미지 |
| 고해상도 | 저해상도/흐릿한 이미지 |
| 구김 없이 펴진 상태 | 구겨진 상태 |

## Settings 탭 - 생성 설정

### 모델 스타일
생성될 모델(사람)의 전체적인 촬영 분위기를 결정합니다.

| 스타일 | 설명 |
|--------|------|
| **아이폰 내추럴** | 실제 아이폰으로 찍은 듯한 자연스러운 느낌 (권장) |
| **스튜디오** | 전문 스튜디오에서 조명을 세팅한 느낌 |
| **캐주얼** | 일상에서 편하게 찍은 스냅 느낌 |

### 배경 스타일
이미지의 배경 장소와 분위기를 결정합니다.

| 스타일 | 설명 |
|--------|------|
| **방 코너** | 자연광이 들어오는 집 안 |
| **크림색 벽** | 깔끔한 무지 배경 |
| **카페** | 아늑한 분위기의 카페 |
| **거울샷** | 전신거울 앞 셀피 스타일 |
| **아웃도어** | 야외 거리 촬영 느낌 |

### 포즈 선택
생성할 이미지의 모델 포즈를 선택합니다. **여러 개 동시 선택 가능**합니다.

| 포즈 | 설명 |
|------|------|
| **정면** | 앞에서 바라본 기본 포즈 |
| **측면** | 옆에서 바라본 프로필 샷 |
| **뒷면** | 뒤에서 바라본 백샷 |
| **연출** | 자연스러운 움직임이 있는 포즈 |
| **디테일** | 의류 디테일을 강조한 클로즈업 |

> 💡 **팁**: 다양한 포즈를 선택하면 룩북의 완성도가 높아집니다!

### 포즈당 컷 수
각 포즈별로 생성할 이미지 개수입니다. (1~10)

| 컷 수 | 용도 |
|-------|------|
| **1~2컷** | 빠른 테스트용 |
| **3~5컷** | 기본 룩북 제작용 (권장) |
| **6~10컷** | 다양한 선택지가 필요할 때 |

> ⚠️ 컷 수가 많을수록 생성 시간이 길어지고 비용이 증가합니다.

### 시드 값 (선택사항)
AI가 이미지를 생성할 때 사용하는 난수의 시작점입니다.

| 상태 | 결과 |
|------|------|
| **비워두면** | 매번 완전히 다른 이미지 생성 |
| **같은 숫자 입력** | 비슷한 구도/스타일의 이미지 재현 가능 |

> 💡 **팁**: 마음에 드는 결과가 나오면 해당 시드 값을 메모해두세요!

### 네거티브 프롬프트 (고급 설정)
생성 결과에서 **제외**하고 싶은 요소들을 입력합니다.

**기본 예시:**
```
blurry, low quality, distorted hands, bad anatomy, watermark
```

> 💡 기본값을 그대로 사용하셔도 충분합니다.

## Prompt 탭 - 프롬프트 설정

### 프롬프트란?
AI에게 어떤 이미지를 생성할지 알려주는 **텍스트 명령어**입니다.
프롬프트가 상세할수록 원하는 결과물에 가까운 이미지가 생성됩니다.

### 템플릿 모드 vs 커스텀 모드

| 모드 | 대상 | 설명 |
|------|------|------|
| **템플릿 모드 (OFF)** | 초보자 | 미리 최적화된 프롬프트 템플릿에서 선택 |
| **커스텀 모드 (ON)** | 고급 사용자 | 영어로 직접 프롬프트 작성 |

> 💡 처음 사용하신다면 **템플릿 모드**를 권장합니다!

### 템플릿 카테고리

| 카테고리 | 설명 |
|----------|------|
| **스타일** | 촬영 스타일 관련 (아이폰 자연광, 스튜디오 클린 등) |
| **모델** | 모델 특성 관련 (한국 모델 등) |
| **배경** | 배경 스타일 관련 (미니멀 배경, 야외 자연 등) |

### 스타일 수식어
기본 프롬프트에 **추가적인 스타일**을 더해주는 옵션입니다.
**여러 개 동시 선택 가능**합니다.

| 수식어 | 효과 |
|--------|------|
| **고화질** | 더 선명한 이미지 |
| **소프트 라이팅** | 부드러운 조명 효과 |
| **에디토리얼** | 잡지 화보 느낌 |

### 커스텀 프롬프트 작성법 (고급)
영어로 원하는 이미지를 구체적으로 설명합니다.

**좋은 예시:**
```
young Korean woman wearing the uploaded clothing, iPhone photo style,
natural lighting, casual pose, cream wall background, full body shot
```

**프롬프트 구성 요소:**
1. 모델 특성 (young Korean woman)
2. 의류 관련 (wearing the uploaded clothing)
3. 촬영 스타일 (iPhone photo style)
4. 조명 (natural lighting)
5. 포즈 (casual pose)
6. 배경 (cream wall background)
7. 구도 (full body shot)

## Train 탭 - LoRA 모델 학습

### LoRA란?
**Low-Rank Adaptation**의 약자로, 특정 스타일이나 인물을 AI에게 학습시키는 기술입니다.

### 학습 방법
1. **학습 이미지 업로드**: 최소 10장, 최대 50장
2. **모델 이름 입력**: 영문으로 입력 권장
3. **학습 시작**: 약 15~30분 소요

### 좋은 학습 이미지 조건
- 일관된 스타일의 이미지
- 다양한 각도/포즈
- 고해상도
- 최소 10장 이상

## 자주 묻는 질문 (FAQ)

### Q: 이미지 생성에 얼마나 걸리나요?
**A:** 설정에 따라 다르지만, 5컷 기준 약 15~20초 정도 소요됩니다.

### Q: 생성된 이미지의 품질이 안 좋아요.
**A:** 다음을 시도해보세요:
1. 다른 AI 모델로 변경 (Model 탭)
2. 고화질 스타일 수식어 추가 (Prompt 탭)
3. 더 좋은 품질의 의류 이미지 업로드

### Q: 같은 설정인데 결과가 매번 달라요.
**A:** 시드 값을 비워두면 매번 다른 결과가 나옵니다. 일관된 결과를 원하시면 시드 값을 고정하세요.

### Q: 특정 포즈가 잘 안 나와요.
**A:** 프롬프트 탭에서 해당 포즈에 대한 설명을 더 구체적으로 추가해보세요.

### Q: API 키 필요라고 나와요.
**A:** 해당 AI 모델을 사용하려면 API 키 설정이 필요합니다. 관리자에게 문의하세요.

### Q: 업로드한 이미지가 사라졌어요.
**A:** History 탭에서 이전 세션을 불러올 수 있습니다.
